# Assets Placeholder

Die App benötigt folgende Asset-Dateien im `assets/` Verzeichnis:

## Benötigte Dateien

- `icon.png` (1024x1024) - App Icon
- `splash.png` (2048x2732) - Splash Screen
- `adaptive-icon.png` (1024x1024) - Android Adaptive Icon
- `favicon.png` (48x48) - Web Favicon

## Temporäre Lösung

Für die Entwicklung können diese Dateien vorerst fehlen. Expo wird Platzhalter verwenden.

Um echte Icons zu erstellen:
1. Verwende ein Design-Tool (Figma, Canva, etc.)
2. Erstelle ein Icon mit Sonne/Wolken/Regenbogen
3. Verwende optimistische Farben (Blau, Gelb, Weiß)
4. Speichere in den angegebenen Größen

## Design-Richtlinien

- **Farben**: #4A90E2 (Blau), #FDB813 (Gelb), #FFFFFF (Weiß)
- **Stil**: Freundlich, modern, optimistisch
- **Elemente**: Sonne, Wolken, Regenbogen, Wettersymbole
